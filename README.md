# PDF 阅读助手 v2.0

WorkBuddy / CodeBuddy 技能 - 快速提取和分析 PDF 文档内容。

## 功能特性

- 文本提取 - 读取 PDF 全文，支持普通 PDF 和扫描版（OCR）
- 表格提取 - 自动识别并提取 PDF 中的表格，输出 Markdown 格式
- 结构化分析 - 自动生成目录、智能摘要、关键词、字数统计
- 智能问答 - 基于 PDF 内容回答问题
- 多 PDF 对比 - 同时读取多份文档，找出共同主题和差异点
- 批量处理 - 一次处理整个文件夹内的多个 PDF，生成汇总报告
- 图片注记 - 标注每页的图表位置和数量

## 安装

将本仓库克隆到你的 WorkBuddy skills 目录：

git clone https://github.com/guipi888/workbuddy-pdf-reader-assistant.git ~/.workbuddy/skills/pdf-reader-assistant/

或通过 WorkBuddy 的技能市场安装。

## 使用方法

帮我读这个PDF：/path/to/file.pdf
批量分析这个文件夹里的PDF：/path/to/folder/
对比这两份PDF：/path/to/a.pdf 和 /path/to/b.pdf
